# VM & MCP Service

Systemd user services for running the NixOS VM and its MCP server on the **host**.

All commands below run on the **host** machine, not inside the VM.

## Prerequisites

- **Nix** (with flakes enabled) — to build the VM image
- **QEMU** — runs the VM (invoked via the `run-personal-vm` script generated by Nix)
- **Bun** — JavaScript runtime for the MCP server
- **SSH agent** running (`SSH_AUTH_SOCK` set) — used by the MCP server to authenticate to the VM
- **OpenSSH client** (`ssh`, `scp`) — used by MCP tools for file transfer and commands

## Setup

Run from the **host**:

```bash
bash setup.sh
```

This will:
1. Install MCP server dependencies (`bun install` in `tools/vm-mcp`)
2. Copy service files to `~/.config/systemd/user/`
3. Patch the project path in the service files
4. Reload systemd and enable both services

## Start

```bash
systemctl --user start vm
systemctl --user start vm-mcp
```

`vm-mcp` depends on `vm` and will start after it automatically.

## Stop

```bash
systemctl --user stop vm-mcp
systemctl --user stop vm
```

## Logs

```bash
journalctl --user -u vm -f
journalctl --user -u vm-mcp -f
```

## Architecture

```
Host (systemd user services)
├── vm.service
│   └── QEMU NixOS VM (4GB RAM, 4 cores)
│       └── SSH on :22 → forwarded to host :2222
└── vm-mcp.service (After=vm.service)
    └── MCP HTTP server on :8765
        └── Connects to VM via SSH (localhost:2222, user joao, SSH agent auth)
```

### vm.service

Runs the NixOS VM via QEMU using the `run-personal-vm` script from the flake build output. The VM disk image is stored at `personal.qcow2` in the project root. SSH inside the VM is forwarded to `localhost:2222` on the host via `QEMU_NET_OPTS`.

### vm-mcp.service

Runs the MCP server (`tools/vm-mcp`) using Bun. It exposes an HTTP endpoint at `http://localhost:8765/mcp` with the following tools for AI agents:

| Tool | Description |
|------|-------------|
| `vm_exec` | Execute a command inside the VM via SSH |
| `vm_status` | Check if the VM is reachable |
| `vm_restart` | Restart the VM service on the host |
| `vm_logs` | Fetch journalctl logs from inside the VM |
| `vm_copy_to` | Copy a file from host to VM via scp |
| `vm_copy_from` | Copy a file from VM to host via scp |

### Environment variables

| Variable | Default | Used by |
|----------|---------|---------|
| `VM_SSH_PORT` | `2222` | vm-mcp |
| `VM_SSH_USER` | `joao` | vm-mcp |
| `MCP_PORT` | `8765` | vm-mcp |
| `NIX_DISK_IMAGE` | `~/proj/angst/personal.qcow2` | vm |
| `QEMU_NET_OPTS` | `hostfwd=tcp::2222-:22` | vm |
